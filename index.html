Help

  • Each line in the text box represents a note
  • Notes can be entered as a frequency ("440Hz"), as a key ("C#4"), or as a ratio of to another note entered earlier in the list.
  • For entering as a ratio, use the format "(index) * ratio" or "(index) * n/d". Index represents the line number of the reference note. "ratio" can be any floating point number. Or you can enter ratio as a fraction "n/d" where n and d are integers.
  • For better understanding, see the presets.
